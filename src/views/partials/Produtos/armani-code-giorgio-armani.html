SOBRE O PRODUTO

Mistério, Moderno, Irresistível. Uma fragrância sofisticada e sedutora para um homem carismático.Armani Code é uma fragrância estrela para quem é Vip, extremamente seletiva, reservada para aqueles do universo privado.

A primeira colônia oriental de Giorgio Armani, Armani Code combina um cocktail de cidra de aroma de bergamota e limão com notas macias de flor de oliveira, aquecido com Guaiac e Tonka Bean. Uma fragrância Eau de Toilette sedutora para o homem contemporâneo.

Uma fragrância inesquecível, Armani Code incorpora a visão de uma fragrância clássica reinterpretada com um toque moderno. Esta fragrância tem um toque atemporal e é como nenhuma outra. Notas: Orientais amadeiradas misturadas com fava tonka, tabaco louro, e madeira de guaiaco.
